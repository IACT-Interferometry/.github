# Welcome to IACT-Interferometry 👋

**IACT-Interferometry** is an open-source organization dedicated to intensity interferometry with Imaging Atmospheric Cherenkov Telescopes (IACTs).

## 🔭 About Us

We develop software tools and analysis pipelines to enable stellar intensity interferometry using the large mirror arrays of IACT systems such as H.E.S.S., MAGIC, VERITAS, and CTA. By exploiting the nanosecond-level timing resolution of these instruments, we aim to measure the angular diameters and surface features of stars at unprecedented resolution.

## 🚀 What We Work On

- **Data analysis pipelines** for processing IACT intensity interferometry observations
- **Simulation tools** for modelling interferometric signals from astrophysical sources
- **Calibration utilities** for baseline characterisation and timing corrections
- **Science applications** including stellar angular diameter measurements
